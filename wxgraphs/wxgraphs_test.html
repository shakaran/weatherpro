Please note that some of these graphs have been renamed since the original set were created. So you will need to check your coding on your website when uploading these graphs to it. There are also more graphs now than before - so choose the ones you find most useful. They also have the capability to change the graph labels to your own language (see Readme.txt).

It is necessary to install all your graphs that you use from this package together with the graphsconf.php file in order for the graphs to work, previous versions of the graphs will not work alongside these.

There have been a number of additions to the user customisable variables in the config file. You do not need to change many of these from the defaults to get the graphs working. The major ones are things like the units being used (USA or Metric) and associated conversion factors and the 12/24 hour setting. You must ofcourse check and if required update the directories for your clientraw files and JPGraph installation.

These graphs have been tested with both PHP4 and JPGraph 1.20.5 - and with PHP5 and JPGraph 2.3.
